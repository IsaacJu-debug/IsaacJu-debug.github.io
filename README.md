# Isaac Ju - Personal Website

Personal portfolio website showcasing research in Graph Neural Networks, High-Performance Computing, and AI-driven simulations for energy applications.

**Live site:** [IsaacJu-debug.github.io](https://IsaacJu-debug.github.io)

## Tech Stack

- Vanilla HTML, CSS, JavaScript
- Markdown content
- GitHub Pages hosting

## Development

```bash
# Serve locally
python -m http.server 8000 -d docs/
```

Then open http://localhost:8000 in your browser.

## Structure

- `docs/index.html` - Main page
- `docs/about.md` - About me section
- `docs/news.md` - Latest news and updates
- `docs/publications.md` - Research publications
- `docs/resume.md` - Education and experience
- `docs/styles.css` - Styling
- `docs/script.js` - Functionality
